# ord()

- 描述

ord() 函数是 chr() 函数__（对于 8 位的 ASCII 字符串）__的配对函数，它以一个字符串（Unicode 字符）作为参数，返回对应的 ASCII 数值，或者 Unicode 数值。

- 语法

ord(str)

- 参数

字符

- 返回值

对应的十进制整数。

## 简单范例


```python
ord('a')
```




    97




```python
ord('b')
```




    98




```python
ord('A')
```




    65




```python
ord("$")
```




    36
